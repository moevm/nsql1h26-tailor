## Как установить зависимости?

```sh
pnpm i
```

## Как запустить?

Вариант 1 (для разработки, через tsx):

```sh
pnpm start
```

ИЛИ:

Вариант 2 (сборка и запуск через tsc и node):

```sh
pnpm build
```

```sh
pnpm preview
```
